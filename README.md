# 100-Days-of-DevOps

A collection of hands-on DevOps, Cloud, and Infrastructure automation tasks.

## Structure

### 100 Days of Cloud (AWS)
- RDS snapshots and restoration
- Lambda functions via CLI
- Load balancers

### 100 Days of Cloud (Azure)
- Disk storage management
- Azure Container Registry (ACR)
- Azure SQL Database
- Web application deployment
- Storage lifecycle management
- MySQL on VMs
- VM security and SSH access
- Container synchronization

### 100 Days of DevOps
- Git operations (remotes, revert, hooks, clone)
- Docker (images, containers, networks, Dockerfile, exec)

### Ansible
- Blockinfile and Unarchive modules
- Soft links creation
- ACL management

### Kubernetes
- Pod deployment and high availability
- Nginx web server deployment
- Shared volumes and sidecar containers
- ReplicationController
- Troubleshooting volume mounts
